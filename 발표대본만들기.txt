mongodb visit name 초기화

- 이 프로젝트를 기획한 의도 ** 어렵다...
1. 와인을 검색하여 사용자에게 정보를 제공해주고싶은 마음에 이 프로젝트를 기회하였다.
2. 내가 먹었던 와인이 언제 수입되고 어디서 만들어졌는지 궁금해서 만들어진 사이트
3. 친구에 의해서 한국에 수입된 와인들을 찾고 더 많은 정보를 제공해주는 사이트를 만들어보고싶었다.
4. 먹어보고싶은 와인이 있지만 한국에 수입되었는지 궁금한 사람들을 위해 만들어진 사이트
5. 한국에 수입되는 와인들을 검색하여 수입일을 기준으로 이름이나 제조국 같은 와인에 대한 정보들을 알려주는 사이트

- 사용한 언어
html, css, javascript(jquery), python


- 소개할 기능
main
	사용자의 ip를 이용하여 이름을 기억하고 방문기록을 남기는 기능
	+ 즐겨찾기 기능도 하려고 했지만 시간상 못함
	시작하면 나오는 로고 애니메이션
	배경화면 6초마다 랜덤으로 바뀌는 기능
	검색창 추가, 검색주제 바꾸는 기능
	검색창에 검색할 값이 있으면 find, 없으면 wine 으로 바뀌는 로고 애니메이션

result
	홈버튼
	맨 위로 버튼
	스크롤 내리면 검색창 숨기기
	스크롤 올리면 검색창 보이기
	페이지 버튼 구현(+wheel)
	+ 결과 클릭하면 검색하는 기능 시간상 못함
